HelloOS
=======

 The HelloOS is written in assembly language.
 HelloOS는 오직 어셈블리어로만 작성됩니다.
 C/C++은 일체 사용되지 않으며 오로지 NASM과 Makefile 만을 이용하여 제작 됩니다.

 해당 운영체제는 USB에 설치되는 구조이며 부팅이 가능하도록 되어있습니다.
 운영체제는 현재 8GB, 16GB USB에 한정적으로 설치가 가능하도록 되어있으며 설치 과정은 추후 작성하여
 본 페이지에 올려지게 됩니다.

 운영체제를 USB에 담은 상태로 타 운영체제에서 일반 USB처럼 사용이 가능하며 추후 USB로 부팅해야만 볼 수 있는
 문서 형식으로 개발해 볼 생각입니다.
